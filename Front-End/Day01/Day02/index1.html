HTML
HTML (HyperText Markup Language) is a standard markup language used to create and design web pages. It structures the content by defining headings, paragraphs, links, images, tables, and other elements. HTML forms the backbone of web pages, while CSS and JavaScript are used for styling and functionality, respectively.
CSS
CSS (Cascading Style Sheets): CSS ek stylesheet language hai jo HTML elements ko style aur design karne ke liye use hoti hai. Isse aap text ka color, font, size, background, layout, spacing, aur animation set kar sakte ho. CSS web pages ko attractive aur visually organized banata hai.
BOOTSTRAP
Bootstrap: Bootstrap ek free and open-source front-end framework hai jo HTML, CSS, aur JavaScript ka use karke responsive aur mobile-friendly websites banane ke liye use hota hai. Isme pre-designed components, grid system, buttons, forms, navigation bars aur modals available hote hain, jisse website development fast aur easy ho jata hai.
JAVASCRIPT
JavaScript: JavaScript ek programming language hai jo web pages me interactive aur dynamic functionality add karne ke liye use hoti hai. Isse aap buttons, forms, animations, sliders, pop-ups aur real-time updates create kar sakte ho. JavaScript client-side aur server-side dono par kaam karta hai.
